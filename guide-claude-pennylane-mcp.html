01
Le protocole MCP

Comment Claude se connecte à Pennylane

MCP (Model Context Protocol) est un protocole ouvert créé par Anthropic, désormais piloté par la Linux Foundation. C'est une prise universelle qui permet à Claude d'agir directement dans vos outils métier — au lieu d'attendre que vous lui copiez-colliez des données.

🧠
Claude (Host)
L'IA qui reçoit vos instructions en langage naturel et décide quelles actions déclencher
🔌
Serveur MCP
Le pont qui expose les 87 fonctionnalités de l'API Pennylane V2 sous forme d'actions que Claude peut déclencher
📊
Pennylane (API V2)
Vos données comptables réelles — factures, écritures, balance, lettrage, dossiers clients
En clair

Sans MCP, Claude est un assistant très compétent mais enfermé dans une pièce — vous devez lui apporter chaque document en main propre. Avec MCP, il a accès à votre bureau, vos classeurs et votre messagerie. Il peut aller chercher l'information directement dans Pennylane, croiser des données de plusieurs dossiers, et exécuter des actions à votre demande.

02
Cas d'usage 1

Interroger plusieurs dossiers en un seul prompt

L'un des cas d'usage les plus puissants : croiser les données de plusieurs clients ou entités simultanément — sans jongler entre onglets, sans copier-coller.

Contexte

Un DAF groupe gère 4 filiales sur Pennylane. Chaque lundi, il passe 2h à consolider les balances et identifier les anomalies. Avec Claude + MCP, c'est une question de secondes.

Prompt exemple
"Compare la balance âgée clients des dossiers Filiale A, Filiale B et Filiale C. Identifie les créances de plus de 60 jours dans chaque dossier et classe-les par montant décroissant."
→ Claude interroge les 3 dossiers en parallèle, consolide les données et retourne un tableau comparatif structuré en quelques secondes.
Prompt exemple
"Pour tous mes dossiers clients, liste les factures fournisseurs en attente de paiement dont l'échéance est dépassée de plus de 15 jours. Groupe par fournisseur commun entre dossiers."
→ Claude identifie les fournisseurs communs à plusieurs dossiers, ce que l'interface Pennylane ne permet pas nativement.
−2h de consolidation manuelle par semaine remplacées par quelques prompts — sans quitter Claude Desktop.
03
Cas d'usage 2

Détection d'anomalies & réconciliation CRM / Compta

Claude peut croiser vos données Pennylane avec d'autres sources (CRM, ERP, tableurs) pour détecter des incohérences que vos outils séparés ne voient pas.

Contexte

Une ESN constate régulièrement des écarts entre les commandes enregistrées dans son CRM et les factures dans Pennylane. Trouver ces écarts prenait 3h par mois à un contrôleur de gestion.

Prompt exemple
"Voici la liste des commandes validées ce mois-ci dans notre CRM [données collées]. Compare avec les factures clients émises dans Pennylane sur la même période. Liste les commandes sans facture correspondante et les factures sans commande associée."
→ Claude croise les deux sources, identifie les écarts et retourne une liste structurée prête à traiter.
Prompt exemple — Détection d'anomalies ANC 2026
"Analyse les écritures de janvier dans le journal des achats. Identifie les imputations qui ne respectent pas les règles ANC 2026 et propose les corrections à apporter."
→ Claude applique les règles ANC intégrées, signale les non-conformités avec explication et suggestion de correction.
−3h de réconciliation manuelle par mois — et une conformité ANC 2026 vérifiée automatiquement à chaque session.
04
Cas d'usage 3

Rédaction d'e-mails clients ultra-contextualisés

Claude peut lire le contexte comptable d'un client directement dans Pennylane et rédiger une communication parfaitement adaptée à sa situation réelle — sans copier-coller de données.

Contexte

Un expert-comptable gérant 40 dossiers passait 1h par semaine à rédiger des emails de relance ou de bilan personnalisés. Avec Claude + MCP, chaque email est généré en 10 secondes avec les données réelles du dossier.

Prompt exemple
"Consulte le dossier du client Martin SAS dans Pennylane. Rédige un email de point mensuel pour le dirigeant : résume la situation de trésorerie, les factures en attente de paiement et les prochaines échéances fiscales. Ton professionnel et synthétique."
→ Claude lit les données réelles du dossier et génère un email personnalisé, contextuel et prêt à envoyer.
Prompt exemple — Relance client
"Pour le client Dupont & Associés, consulte les factures impayées dans Pennylane et rédige un email de relance amiable. Mentionne chaque facture concernée avec son montant et sa date d'échéance. Ton cordial mais ferme."
→ Claude génère une relance personnalisée avec les vraies données du dossier — aucun copier-coller nécessaire.
−1h de rédaction par semaine — chaque email généré en 10 secondes avec les données réelles du dossier client.
05
Guide d'installation

Connecter Claude à Pennylane en 5 étapes

Pas besoin d'être développeur. Comptez environ 1 heure pour la configuration initiale. Une fois en place, la connexion est permanente.

🔒
Prérequis : Claude Desktop installé (Mac ou PC) • Python 3.10 ou supérieur • Compte Pennylane avec abonnement Essentiel ou supérieur • Rôle administrateur sur votre espace Pennylane
1
Générer votre token API Pennylane
Connectez-vous à Pennylane → Paramètres → Intégrations → API. Cliquez "Générer un token". Copiez-le immédiatement — vous ne pourrez plus le voir une fois la fenêtre fermée.
2
Installer le serveur MCP Pennylane
Ouvrez votre terminal et installez le serveur MCP via pip :
pip install pennylane-mcp-server
3
Configurer la variable d'environnement
Créez un fichier .env à la racine du projet et ajoutez votre token — jamais en clair dans le code :
PENNYLANE_API_KEY=votre-token-ici
4
Configurer Claude Desktop
Ouvrez le fichier de configuration Claude Desktop et ajoutez le serveur MCP :

Mac : ~/Library/Application Support/Claude/claude_desktop_config.json
Windows : %APPDATA%\Claude\claude_desktop_config.json
{ "mcpServers": { "pennylane": { "command": "pennylane-mcp", "env": { "PENNYLANE_API_KEY": "votre-token-ici" } } } }
5
Redémarrer Claude Desktop et tester
Relancez Claude Desktop. Les outils Pennylane apparaissent sous l'icône marteau 🔨. Testez avec un prompt simple :
"Liste mes 5 dernières factures clients dans Pennylane"
Ressources : Claude Desktop Python 3.10+ pennylane-mcp-server API Pennylane V2 ANC 2026
